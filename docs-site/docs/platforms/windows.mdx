# Windows

Platform-specific information for running Mystral Native.js on Windows.

## System Requirements

- Windows 10 (version 1903) or later
- DirectX 12 capable GPU
- x64 processor

## Recommended Build

```
mystral-windows-x64-v8-dawn.zip
```

**V8 + Dawn** provides the best experience with full ES2024+ JavaScript and Chrome's WebGPU implementation.

## Installation

### Manual Install

1. Download from the [releases page](https://github.com/mystralengine/mystralnative/releases)
2. Extract the zip file
3. Run `mystral.exe` from PowerShell or Command Prompt

```powershell
# Download (PowerShell)
Invoke-WebRequest -Uri "https://github.com/mystralengine/mystralnative/releases/latest/download/mystral-windows-x64-v8-dawn.zip" -OutFile mystral.zip

# Extract
Expand-Archive mystral.zip -DestinationPath mystral

# Run
cd mystral\mystral-windows-x64-v8-dawn
.\mystral.exe --version
```

### Add to PATH

To run `mystral` from anywhere:

1. Extract to a permanent location (e.g., `C:\Program Files\MystralNative`)
2. Add to System PATH:
   - Open System Properties > Advanced > Environment Variables
   - Edit PATH and add the directory containing `mystral.exe`

## Building from Source

### Prerequisites

- **Visual Studio 2022** with C++ Desktop workload
- **CMake 3.20+** (from cmake.org or Visual Studio)
- **Bun** for dependency management

```powershell
# Install bun
powershell -c "irm bun.sh/install.ps1 | iex"
```

### Build Commands

```powershell
git clone https://github.com/mystralengine/mystralnative.git
cd mystralnative

bun install
bun run deps:download
bun run configure
bun run build

.\build\Release\mystral.exe --version
```

### Visual Studio Generator

```powershell
cmake -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Available Configurations

| JS Engine | WebGPU Backend | Notes |
|-----------|----------------|-------|
| V8 | Dawn | Recommended, best compatibility |
| V8 | wgpu-native | Alternative backend |
| QuickJS | Dawn | Smaller binary |
| QuickJS | wgpu-native | Smallest binary |

**Note**: JavaScriptCore (JSC) is not available on Windows.

## GPU Drivers

Ensure you have up-to-date GPU drivers:

- **NVIDIA**: [nvidia.com/drivers](https://www.nvidia.com/Download/index.aspx)
- **AMD**: [amd.com/drivers](https://www.amd.com/en/support)
- **Intel**: [intel.com/drivers](https://www.intel.com/content/www/us/en/download-center/home.html)

## Known Issues

### Missing DLLs

If you see "missing DLL" errors, install the Visual C++ Redistributable:

[Download VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### WebGPU Not Available

If WebGPU initialization fails:

1. Update GPU drivers
2. Ensure Windows 10 version 1903 or later
3. Check that DirectX 12 is available

### Antivirus Warnings

Some antivirus software may flag unsigned executables. Add an exception for the Mystral Native.js directory.

## Performance Tips

- **Fullscreen**: Exclusive fullscreen may improve performance
- **V8 JIT**: Provides significant speedup for complex JS
- **Dawn**: Generally better Windows support than wgpu-native
